# 更新日志

## v1.2.1 (2024-01-12)
### 修复
- 修复复制功能复制错误内容的问题
- 改进 GitHub URL 提取逻辑，正确处理嵌入在描述文本中的 URL
- 标准化 GitHub URL 格式为 `https://github.com/owner/repo`

### 新增
- 添加调试工具页面（debug.html）用于测试和缓存管理
- 添加缓存查看和清除功能
- 添加单个应用测试功能

### 改进
- 增强 URL 解析，支持更多格式
- 改进深度搜索算法
- 添加故障排除文档

- 支持使用 `{github_url}` 占位符自定义复制内容
